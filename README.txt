# Two Portfolios Website

This is a simple static website that showcases two separate portfolios — **Anush Jowin A** and **Harshdeep Sharma**.  
Each portfolio includes an About section, Skills, Projects, and Contact details, all presented in a clean dark theme with neon highlights.

## 🌐 Live Demo
[Click here to view](https://github.com/anushjowin/Personal-Portfolio)  

## 📂 Project Structure
two-portfolios/
│
├── index.html # Main HTML file
├── /assets # Images and other assets
│ ├── Anush.jpg
│ └── harsh.jpg
└── README.md # Documentation

## ✨ Features
- Two portfolios on a single webpage
- Dark theme with neon border effects
- Separate navigation menus for each portfolio
- Sections: About, Skills, Projects, Contact

## 🛠 Technologies Used
- HTML5
- CSS3

## 🚀 How to Run
1. Navigate to the project folder:
   
   cd Two Portfolio
2. Open 'index.html' in your web browser.
